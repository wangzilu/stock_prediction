## 思路：三条线并行

### 线路 1：新因子注入（最高 ROI，1 周见效）

**逻辑：** 当前 Alpha158 只有价量因子（价格、成交量、技术指标）。A 股市场中，资金流和机构行为是独立于价量的 alpha 来源。两个维度低相关 = 组合后 IC 提升大。

```
当前：Alpha158 (158 维价量因子) → XGB → IC=0.024

目标：Alpha158 (158维) + 资金流 (5-10维) + 北向 (3-5维) → XGB → IC=0.04+
```

**具体步骤：**

| 天 | 任务 | 产出 |
|---|------|------|
| Day 1 | `data/collectors/fund_flow.py` — 采集主力资金流 + 北向资金 | 每日 5000+ 只的资金流数据 |
| Day 2 | `factors/fund_factors.py` — 计算 5 个资金因子 | mfd_sellord_z, net_inflow_ratio, northbound_pct_change, northbound_z, flow_momentum |
| Day 3 | 写入 Qlib 自定义特征 bin，和 Alpha158 合并 | 163 维特征（158+5）|
| Day 4 | XGB 重训 + 评估 | 对比 IC：158 维 vs 163 维 |
| Day 5 | 如果 IC 提升，更新生产模型 | 推文用增强版模型 |

**为什么这个最优先？**
- 华泰研报回测 mfd_sellord 因子独立 Sharpe 4.43
- 中金研报回测北向资金因子独立 Sharpe 2.46
- 两者和价量因子相关性低（不同维度）
- AKShare 免费日频数据，无积分限制
- 幻方量化的因子构成中 40% 价量 + 20% 替代数据，我们缺的就是这 20%

### 线路 2：模型工程化（中 ROI，提升稳定性）

**逻辑：** 当前模型单次训练，没有滚动验证、没有归因、没有实验对比基线。即使因子好了，没有工程化就不知道"好了多少"和"什么时候变坏了"。

```
当前：train_lgb.py → lgb_model.pkl → 推文
           ↑ 没有对比、没有归因、没有退化检测

目标：RollingGen → 多期训练 → Brinson 归因 → IC 退化预警 → 自动降级
```

**具体步骤：**

| 天 | 任务 | 产出 |
|---|------|------|
| Day 1 | `pip install statsmodels` + Qlib 报告模块 | IC/收益可视化 |
| Day 2 | 用 Qlib `RollingGen` 替代自写 `rolling_train.py` | 原生滚动训练 |
| Day 3 | 接入 `SignalRecord` + `Recorder` | 每次训练自动记录 pred/label/IC |
| Day 4 | `brinson_pa()` 归因 | 知道 alpha 来自行业配置还是选股 |
| Day 5 | IC 退化预警 — 连续 3 天 IC < 0 自动降级 | 推文标注"模型信号减弱" |

**为什么重要？**
- 灵均/衍复的核心竞争力不只是因子多，而是**因子治理** — 知道哪个因子在衰减、什么时候该换
- Lopez de Prado 反复强调：没有 walk-forward 验证的回测结果都是假的
- 当前 rolling 用自写脚本（cc），Qlib 的 `RollingGen` 更完善（支持 expanding/sliding/trunc）

### 线路 3：情绪+行为因子（中长期，差异化 alpha）

**逻辑：** 头部量化私募已经用深度学习挖完了价量因子，价量维度的 alpha 正在衰减。下一个 alpha 来源是**行为数据**（游资席位、涨停板情绪、社交媒体扩散）和**事件冲击**（政策/监管/公告）。

```
当前：RSS 新闻 → MiniMax LLM → 地缘因子（宏观级）

目标：涨停溢价率 + 龙虎榜游资 + 股吧情绪 + 公告事件
      → 结构化事件表 (event_impacts)
      → 个股级情绪因子 + 行业级热度因子
      → 喂入 XGB 或作为 SignalScorer 权重修正
```

**具体步骤：**

## 优先级排序 — 一句话版

> **先加数据（资金流），再加工程（滚动+归因），最后加行为（情绪+事件）。**

原因：
1. 加数据是**乘法效应** — 新维度的因子和现有因子低相关，IC 提升是乘法
2. 加工程是**除法效应** — 防止虚高，挤掉水分，知道真实 alpha 是多少
3. 加行为是**加法效应** — 逐步积累，需要时间建数据集和验证

## 当前系统 vs 头部私募的差距

| 维度 | 当前系统 | 百亿私募 | 差距 |
|------|---------|---------|------|
| 因子维度 | 158（纯价量）| 500+（价量+基本面+替代+资金流）| **缺资金流和替代数据** |
| 模型 | XGB 单模型 | DL ensemble + 非线性因子生成 | 中等（XGB 效果不差）|
| 训练方式 | 单次训练 | 滚动+在线+元学习 | **缺滚动和退化检测** |
| 数据更新 | 日频盘后 | 分钟/tick 级 | 大（但日频已够用做推荐）|
| 风控 | 简单止盈止损 | 组合级风控+因子暴露控制 | 中等 |
| 执行 | 推荐不执行 | 毫秒级自动交易 | 不可比（定位不同）|

**关键洞察：差距最大的不是模型（XGB 已经接近 3-5 年前的百亿水平），而是因子维度。加 5-10 个资金流因子可能比换 Transformer 提升更大。**
