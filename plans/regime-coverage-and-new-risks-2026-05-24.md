## 3. 未来潜在大跌大涨因素调研

### 已经历过的不再重复。以下是**以前没出现过但理论上可能发生的**：

### 3.1 A 股特有的新风险

**3.1.1 量化策略极度拥挤后的"算法共振崩盘"**

2024 微盘踩踏已经是预演。但更极端的版本是：
- A 股量化私募 AUM 已超 2 万亿
- 大量策略使用相似因子（Alpha158 类价量因子）
- 如果某个触发事件（监管限制转融通、雪球敲入、ETF 集中赎回）导致同时平仓
- 可能出现"所有量化策略同一天卖同一批股票"的极端踩踏
- 2024 只是微盘，下次可能蔓延到中盘甚至大盘

**前置信号**：量化产品超额收益同步回撤、IC/IM 基差急剧扩大、同类因子拥挤度指标
**我们能监控吗**：❌ 缺 IC/IM 基差数据。部分可用 limit_list_d 代替

**3.1.2 地方债务危机传导到 A 股**

- 地方政府融资平台（LGFV）债务问题一直存在
- 如果某个省级城投平台实质违约 → 信用市场恐慌 → 银行股暴跌 → 传导到全市场
- 这在 A 股从未发生过（信用债违约一直被控制在小范围）

**前置信号**：城投债利差飙升、信用债取消发行激增、地方银行股异常下跌
**我们能监控吗**：⚠️ Shibor/信用利差部分覆盖。LLM 新闻可以抓到"违约"关键词

**3.1.3 中美科技脱钩升级 — 金融制裁**

- 当前贸易战主要是关税，金融制裁还没大规模实施
- 如果美国对中国头部科技公司实施 SDN 制裁（类似华为但更广泛）
- 或限制美国资本投资中国科技股（已有先例）
- 可能导致外资集中撤离 + 科技股暴跌

**前置信号**：美国国会听证会、行政令草案泄露、外交部措辞升级
**我们能监控吗**：✅ LLM 新闻 pipeline 可以抓到。但 BBC/RSS 的权重已经降到 15%

**3.1.4 A 股 T+0 / 注册制 / 做空机制 重大改革**

- 如果突然宣布 T+0（当天买卖）→ 短期大幅波动
- 或者大幅放开做空 → 空头力量释放
- 历史上 A 股每次交易制度改革都引发剧烈波动（2010 股指期货、2014 沪港通、2019 科创板）

**前置信号**：证监会/上交所征求意见稿、两会提案、媒体吹风
**我们能监控吗**：✅ LLM 新闻 + 东财公告

### 3.2 全球层面可能传导到 A 股的新风险

**3.2.1 日本加息引发全球 Carry Trade 平仓**

2024.08 已经预演过一次（日经 -12.4%），但 A 股当时影响不大。如果日本持续加息：
- 全球借日元投资高收益资产的 carry trade 大规模平仓
- 可能导致全球权益资产同步下跌
- A 股通过港股通/北向资金通道受影响

**前置信号**：日元急速升值 + VIX 飙升 + 全球股市同步下跌
**我们能监控吗**：⚠️ 纳指/恒指在监控，但日元/日经不在

**3.2.2 AI 泡沫破裂**

- 2025-2026 全球 AI 投资热潮可能形成泡沫
- 如果某个标志性事件（AI 大模型安全事故、GPU 产能过剩、头部 AI 公司财报不及预期）
- 导致全球科技股暴跌 → 传导到 A 股科技/AI 板块

**前置信号**：纳斯达克科技巨头财报、GPU/半导体库存周期、AI 应用商业化进度
**我们能监控吗**：✅ LLM 新闻 + 纳指回调

**3.2.3 全球能源危机 2.0**

- 中东冲突升级（伊朗/以色列）→ 霍尔木兹海峡封锁 → 油价 > $150
- 或台海冲突导致海运中断
- 输入性通胀 → 全球央行被迫加息 → 股市暴跌

**前置信号**：油价急涨、运费指数飙升、地缘新闻密度
**我们能监控吗**：⚠️ LLM 新闻有但权重已降。原油数据没有

**3.2.4 网络攻击/交易所系统故障**

- 2024 CrowdStrike 事件导致全球 IT 中断
- 如果 A 股交易所核心系统被攻击或故障 → 交易暂停 → 恐慌
- 或者重大数据泄露事件影响金融基础设施

**前置信号**：几乎无法提前预测（黑天鹅中的黑天鹅）
**我们能监控吗**：❌ 这种事件只能事后反应

### 3.3 经济学理论可预见的风险

**3.3.1 明斯基时刻（Minsky Moment）**

- 长期稳定导致过度冒险 → 杠杆累积 → 某个触发点引发连锁去杠杆
- A 股 2015 就是经典 Minsky Moment
- 下一次可能来自：量化杠杆（DMA/转融通）、地产债务、地方债

**前置信号**：杠杆率持续上升 + 波动率极低 + 投机热度极高
**我们能监控吗**：✅ leverage_score + volatility(STD20) + theme_breadth

**3.3.2 流动性陷阱（Liquidity Trap）**

- 央行持续降息但经济不恢复 → 企业不贷款不投资 → "推绳子"困境
- 日本 1990 年代的翻版
- 对 A 股：估值中枢持续下移，长期阴跌

**前置信号**：M1/M2 剪刀差持续扩大、贷款需求低迷、PPI 持续为负
**我们能监控吗**：✅ M1/M2 + Shibor + PMI

**3.3.3 通缩螺旋**

- 物价下跌 → 消费推迟 → 企业利润下降 → 裁员 → 消费更弱 → 物价继续跌
- 2024-2025 中国 CPI 在 0 附近，PPI 持续为负，有通缩苗头

**前置信号**：CPI < 0 连续 3 月 + PPI < -3% + 社融增速下降
**我们能监控吗**：✅ inflation_score 用的就是 CPI
