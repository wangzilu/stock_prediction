## Phase 1：低垂果实（第 1-2 周）— +80 因子

### 1A. 估值因子 — 一天搞定，IC 增量最大

```python
# AKShare 一行就能拿到 PE/PB/PS/总市值/流通市值
import akshare as ak
df = ak.stock_a_indicator_lg(symbol="000001")
# 返回: pe_ttm, pb, ps_ttm, total_mv, circ_mv, dv_ratio(股息率)
```

| 因子 | 公式 | 证据 |
|------|------|------|
| PE_TTM | 市价/每股收益 | Fama-French 价值因子核心，A 股 IC ~0.03-0.05 |
| PB | 市价/每股净资产 | 经典 HML 因子 |
| PS_TTM | 市价/每股营收 | 亏损股比 PE 更好用 |
| 股息率 | 年分红/股价 | 高股息策略在 A 股稳定 |
| EP (盈利收益率) | 1/PE | 分布更对称 |
| PEG | PE/利润增速 | 成长+价值结合 |
| PE 分位数(3 年) | 当前 PE 在 3 年历史中排多少 | 相对估值 |

**接入方式：** 每周跑一次，存到 `data/storage/fundamental_features.parquet`，训练时和 Alpha158 按 `(date, instrument)` join。

### 1B. 质量因子 — 两天搞定

```python
# 财务分析指标
df = ak.stock_financial_analysis_indicator_em(symbol="000001")
# 返回: ROE, ROA, 毛利率, 净利率, 资产负债率, 流动比率, 速动比率...
```

| 因子 | 含义 | 为什么有效 |
|------|------|-----------|
| ROE_TTM | 净资产收益率 | 巴菲特最爱，衡量赚钱效率 |
| ROA_TTM | 总资产收益率 | 资产效率 |
| 毛利率 | 定价权力 | 高毛利 = 护城河 |
| 净利率 | 最终盈利深度 | |
| 资产负债率 | 杠杆风险 | 高杠杆 = 高风险 |
| 应计利润率 | (净利-经营现金流)/总资产 | 盈利质量，IC ~0.02 |
| Piotroski F-Score | 9 项综合打分 | 强多空因子 |

## 架构设计 — FeatureMerger

```
Alpha158 (日频, 158列) ──────────┐
宏观因子 (日频, 16列) ────────────┤
资金流 (日频, 14列) ──────────────┤──→ FeatureMerger ──→ XGB
基本面 (季频→日频前填, 47列) ──────┤     (按 date×instrument join)
行业因子 (日频, 9列) ──────────────┤
股东因子 (季频→日频前填, 9列) ──────┘

输出: ~253 列特征矩阵
```

**关键设计决策：**
- 基本面/股东因子是季频，用 `forward-fill` 填到日频
- 宏观因子所有股票共享，用 `broadcast`
- XGB 原生支持 NaN，无需特殊填充
- **不改 Qlib 内部**，在模型输入前外部 merge
