# Stock Prediction System Design

## Overview

A股 + 加密货币 + 黄金预测系统，以舆情分析和国际局势作为辅助信号源，每日收盘前1小时推送荐股信号，5日后自动印证，异常舆情/地缘事件时实时风险警示。

## 核心需求

- **目标市场**：A股、加密货币、黄金
- **预测方式**：多周期（短线/中线/宏观）融合评分
- **使用方式**：纯信号提示，企业微信推送，人工决策
- **数据源**：中文社交媒体为主 + 全覆盖国际新闻/政策
- **技术栈**：Python全栈
- **节奏**：MVP优先，快速验证

### 舆情数据

|------|----------|----------|

| 雪球 | 爬虫 | 个股讨论、组合动态 |
| 东方财富股吧 | 爬虫 | 个股情绪、散户观点 |
| 财经新闻 | RSS + 爬虫 | 新浪财经、财联社快讯 |

### 国际/宏观数据

| 数据源 | 内容 | 获取方式 |
|--------|------|----------|

- 国际新闻/GDELT：每小时或事件驱动

## 因子工程层

### 量化因子（Qlib）

| 类别 | 示例因子 | 适用周期 |
|------|----------|----------|
| 技术面 | MA/MACD/RSI/布林带/ATR | 短线/中线 |
| 量价 | 成交量异动、换手率、资金流 | 短线 |
| 动量/反转 | N日收益率、波动率 | 中线 |
| Qlib内置 | Alpha158/Alpha360因子集 | 多周期 |

### 舆情因子

| 因子 | 计算方式 | 信号含义 |
|------|----------|----------|
| 情感得分 | FinGPT中文金融情感模型打分，按标的聚合 | 正面/负面倾向 |
| 舆情热度 | 单位时间内提及量的变化率 | 关注度异动 |
| 大V共识 | 头部KOL观点方向一致性 | 市场共识强度 |
| 情绪背离 | 舆情乐观但价格下跌（或反之） | 潜在反转信号 |

NLP模型：优先用FinGPT中文金融情感模型（LoRA微调，轻量部署）。

### 地缘/宏观因子

| 因子 | 数据源 | 信号含义 |
|------|--------|----------|
| 地缘风险指数 | GDELT冲突/合作事件计数 | 避险情绪→利好黄金 |
| 政策变动信号 | 央行声明关键词检测 | 加息/降息预期 |
| 中美关系温度 | GDELT双边事件Goldstein分数 | A股/加密风险偏好 |
| 制裁/贸易战信号 | 新闻关键词+事件分类 | 板块级别影响 |

### 因子接入方式

舆情和地缘因子处理为日频时序数据，与行情因子对齐存入Qlib本地数据仓库，作为额外特征列参与模型训练。

## 信号决策层

### 多周期模型

| 周期 | 模型 | 输入侧重 | 输出 |
|------|------|----------|------|
| 短线（1-5天） | LightGBM（Qlib内置） | 量价因子+舆情热度/情感突变 | 个股涨跌概率 |
| 中线（1-4周） | LSTM/Transformer | 技术+舆情趋势+资金流 | 趋势方向+强度 |
| 宏观（月级） | 规则引擎+简单ML | 地缘因子+央行政策+避险情绪 | 大类资产配置倾向 |

### 信号融合

```python
signal_score = (
    short_term_score * weight_short +
    mid_term_score * weight_mid +
    macro_score * weight_macro
)

# 信号等级
if signal_score > HIGH_THRESHOLD:    → "强烈看多"
elif signal_score > MID_THRESHOLD:   → "看多"
elif signal_score < -HIGH_THRESHOLD: → "强烈看空"
elif signal_score < -MID_THRESHOLD:  → "看空"
else:                                → "观望"
```

### 冲突处理

- 短线和中线方向矛盾时降低信号等级，标注"信号分歧"
- 宏观发出风险警示时，压制所有看多信号

## 推送机制

### 三种推送场景

| 场景 | 时间 | 内容 |
|------|------|------|
| 每日荐股 | 14:00（收盘前1小时） | 基于当日盘中数据+舆情+宏观，推荐标的及方向 |
| 5日印证 | 推荐后第5个交易日14:00 | 回顾推荐结果：涨跌幅、胜率、模型复盘 |
| 风险警示 | 实时（事件驱动） | 舆情突变/地缘风险升级，建议空仓或降仓 |

### 推送模板

**每日荐股（14:00）**：
```
📈 今日推荐 (日期 14:00)
─────────────
1. 标的名称(代码) | 方向 | 评分 X.X
   理由：简要因子说明
2. ...
─────────────
建议仓位：X成
宏观环境：描述
```

**5日印证**：
```
📋 荐股印证 (推荐日 → 印证日)
─────────────
1. 标的名称(代码) | 推荐方向
   结果：涨跌幅 ✅/❌ | 最高 | 最大回撤
─────────────
本轮胜率：X/Y | 累计胜率：Z%
模型反思：简要分析偏差原因
```

**风险警示（实时）**：
```
⚠️ 风险警示 (时间)
─────────────
触发原因：事件描述
影响评估：影响范围
─────────────
建议：仓位/操作建议
当前持仓影响：受影响标的列表
```

### 推送约束

- 每个标的每日最多推送2次
- 同级别重复信号不推（只推等级变化）
- 支持配置关注列表（自选股/币/金）

## 运行时序

```
13:00  数据采集截止（A股盘中数据+当日舆情+国际新闻）
13:00-13:30  因子计算（量化+舆情+地缘）
13:30-13:50  模型推理 + 信号融合
13:50-14:00  生成推荐报告
14:00  企业微信推送

每小时  舆情/GDELT事件检查 → 异常则触发风险警示
每日15:30  收盘数据归档，更新印证追踪表
第5交易日14:00  自动生成印证报告并推送
```

## MVP范围（第一阶段）

1. A股行情接入 + Qlib Alpha158基线模型跑通
2. 雪球/东财舆情采集 + FinGPT情感打分作为因子
3. LightGBM短线模型 + 简单信号评分
4. 企业微信14:00定时推送
5. 印证追踪模块（记录推荐，5日后自动对账）
